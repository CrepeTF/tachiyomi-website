Creating backups

Backups are inter-compatible between Tachiyomi versions, with a few exceptions listed under forks information.

Backups can hold the following information:

  1. Titles
  2. Categories
  3. Read chapters
  4. Tracking settings
  5. Reading history

You can do this by going to

more_horiz
More →
settings
Settings →
backup
Backup and then selecting Create backup and select a location to save it.

Restoring

Restoring is done from the

backup
Backup settings. To restore without issues be sure to:

  • Install all the EXTENSIONS that were installed at the time of the backup.
  • Log into the
    autorenew
    Tracking     services that you use.
  • Log into MangaDex using WebViewas explained here.
  • You're able to access all the sources through the WebView.
  • Be on a consistent internet connection.

Transferring downloads

You can transfer downloaded manga chapters from one version of Tachiyomi to another.

You can do this by going to

more_horiz
More →
settings
Settings →
get_app
Downloads and then set the download directory to that of the download folder of your old Tachiyomi, then restore the backup.

Turning on auto-backups

It is highly recommended you turn on auto backups, this will ensure you can recover if need be.

You can do this by going to

more_horiz
More →
settings
Settings →
backup
Backup and then setting a backup frequency. In the case of a catastrophic failure, at least you will be able to recover.

Forks information

Neko

This fork can only restore MangaDex entries in a backup. If there are entries from other sources that are a part of that backup then they will not transfer.

TachiyomiAZ, TachiyomiSY and TachiyomiEH

These forks have some built-in extensions, namely: E-Hentai, Hitomi.la, and nHentai for TachiyomiAZ and TachiyomiEH while TachiyomiSY only has E-Hentai.

Titles from said extensions will not be restored when restoring a backup made by these forks to standard Tachiyomi. It is possible to restore titles to these forks, but it is not possible to back them up out of these forks.
